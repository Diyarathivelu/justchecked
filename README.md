# 🧠 JustChecked – Interactive Web Application

## 📌 Description

**JustChecked** is an interactive web application designed to explore and understand basic number theory and string logic through real-time validation tools.

The platform includes multiple logic-based modules such as:
Armstrong Number, Prime Number, Palindrome Checker, Fibonacci Series, Anagram Checker, Pangram Checker, LCM & GCD, and more.

This project is built for **learning, logical thinking improvement, and interview preparation**.

---

## ✨ Key Features

- 💡 Modular logic-based tools with definition, example, and real-time input/output
- ⚡ Interactive frontend using **HTML, Bootstrap, and jQuery (AJAX)**
- 🔁 Backend processing using **PHP for real-time validation**
- 📊 Includes 15+ logic modules (Prime, Armstrong, Fibonacci, Pangram, Anagram, etc.)
- 🎯 Clean, responsive UI with structured navigation
- 📘 Designed for **logical thinking and interview preparation**

---

## 🛠 Technologies Used
